Privacy & Portability

Blocks common ad/analytics hosts. May break some sites; turn off if a page fails to load.

VPN / proxy routing

PySearch is not a VPN by itself. Route traffic through your VPN’s local proxy (HTTP or SOCKS5), e.g. 127.0.0.1 and the port your app shows. System-wide VPN still protects other apps separately.

Cloud sync (optional)

Run npm run cloud-server from the app repo, then sign in on the dashboard. Vault is AES-encrypted locally before upload.

Updates

Installed app checks your release server (see package.json → build.publish). Set env PYSEARCH_UPDATE_URL to override.
